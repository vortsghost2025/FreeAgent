# Roadmap

This roadmap outlines the near‑term, mid‑term, and long‑term goals for the FreeAgent project and its related ecosystem. It is a living document and will be updated as priorities change.

## Near Term (0–3 months)

- [ ] Complete Phase 1 verification and closeout tasks.
- [ ] Harden Compact Phenotype Continuity Gate (testing and validation; no runtime enablement).
- [ ] Finalize WE4Free coupling and deployment review for future extraction.
- [ ] Conduct PHI/synthetic‑data audit for medical demos.
- [ ] Establish CI pipeline for shared‑infra as an installable package.
- [ ] Update and publish governance documentation (decision records, lane lattice references).

## Mid Term (3–9 months)

- [ ] Phase 2 extraction readiness assessment (pending governance decisions).
- [ ] Implement and test continuity gate (feature‑flagged, still disabled).
- [ ] Refine delegation contracts and constraint discovery pipeline.
- [ ] Expand WE4Free public site coverage (additional regions).
- [ ] Improve developer experience: clearer docs, better templates, issue triage.
- [ ] Integrate more observability into FreeAgent core (metrics, structured logging).

## Long Term (9+ months)

- [ ] Enable continuity gate in production (if validated and approved).
- [ ] Autonomous constraint discovery → lattice ratification workflow.
- [ ] Potential separation of FreeAgent core into independently versioned modules.
- [ ] Strengthen public archive and research outputs (Deliberate‑AI‑Ensemble).
- [ ] Explore federation and mesh resilience patterns at scale.

## Notes

- **No Phase 2 extraction is authorized until explicitly approved.**
- **Runtime enforcement features must remain disabled by default until validated.**
- **Governance authority remains with the 4‑lane lattice; FreeAgent does not assign constitutional authority.**
