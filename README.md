# FreeAgent

FreeAgent is a lightweight, on‑device AI orchestration system designed for stability, speed, and efficient use of limited hardware. It manages multiple agents locally, monitors system load in real time, and automatically adapts behavior to prevent crashes, drift, or slowdowns.

## Features
- Local multi‑agent orchestration
- Real‑time stress scoring (CPU, RAM, latency)
- Automatic mode switching and cleanup cycles
- Stable long‑running sessions
- Works with local LLMs (LM Studio, Ollama, OpenClaw)
- Zero cloud dependency

## Getting Started
1. Clone the repo  
2. Install dependencies  
3. Run the local agent runtime  
4. Connect your preferred local model
