# API Keys for MEV Swarm\n\nThis file stores API keys for the MEV Swarm system.\n\n## Location\n\nAll keys should be stored in: `mev-swarm/.env`\n\n## Required Keys\n\n| Key | Description | Where to Get |\n|-----|-------------|--------------|\n| PRIVATE_KEY | Wallet private key (64 hex chars, no 0x) | Your wallet export |\n| ETH_RPC_URL | Ethereum mainnet RPC | Alchemy or Infura |\n| ETHERSCAN_API_KEY | For contract verification | Etherscan.io |\n| FLASHBOTS_RPC_URL | Flashbots bundle RPC | rpc.flashbots.net |\n\n## Example .env format\n\n```\nPRIVATE_KEY=your_64_char_hex_key_here\nETH_RPC_URL=https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY\nETHERSCAN_API_KEY=your_etherscan_key\nFLASHBOTS_RPC_URL=https://rpc.flashbots.net/fast?originId=your_id\n```\n\n## Security\n\n- Never commit .env to git\n- Keep this file private\n- Keys are loaded via dotenv in Node.js\n
