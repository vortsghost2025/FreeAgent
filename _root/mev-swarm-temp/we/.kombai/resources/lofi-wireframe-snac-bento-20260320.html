📐 Design Notes — Key Decisions

  • Compact Header (52px vs 122px): Saves 70px of vertical space. Multi-service health bar replaces single Backend badge — shows Ollama, Redis, Qdrant, FreeCoder individually.
  • Icon-only Sidebar (64px): Reclaims ~340px of horizontal space for agent panels. Clicking any icon expands to 240px drawer. Each section has its own icon with label — clear for low-vision users.
  • Bento 3-column grid: Col 1 = Free Coding Agent + controls tray + Memory Timeline. Col 2 = SNAC Agent + Swarm Queue + Project Vault. Col 3 = persistent status sidebar (Token Cost, Swarm Monitor, Topology, Intelligence). No hidden panels.
  • Controls Tray (tabbed): Agent Task, Ingest Document, Quick Thought collapsed into a tabbed tray at the bottom of each agent panel — reduces sidebar clutter by 60%.
  • Focus Mode (⛶ button): Each panel has an expand button. Clicking it expands that panel to fill the full bento grid area. All other panels collapse. Clicking again restores the grid.
  • Status Sidebar (right column, spans 2 rows): All monitoring panels (Token Cost, Swarm Monitor, Topology, Intelligence, Shared Knowledge) live here permanently — always visible, always accessible.
  • Timeline Filter Bar: New search + type dropdown for filtering 50+ events — essential for low-vision users who need to find specific agent_start events without reading every row.
  • High Contrast Toggle: Header button that boosts all text to near-white, borders to full brightness, accent to pure electric cyan — one-click vision accessibility mode.
  • Final color palette: Deep Space Navy #0a0f1e bg · Electric Cyan #00E5FF accent · Neural Green #00FF9D connected state · Near-white #F0F9FF text
