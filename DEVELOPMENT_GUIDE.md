# DEVELOPMENT GUIDE

## Project Structure
- `dialogue_engine.py`, `narrative_generator_advanced.py`: Narrative
- `political_system.py`, `multi_federation_politics.py`: Politics
- `persistent_universe.py`, `progression_tracker.py`: Persistence
- `federation_dashboard.*`: UI
- `mobile_extension_stub.py`: Mobile/cross-platform extension

## Contributing
- Write tests for all new features
- Follow modular design
- Document all public APIs

## Testing
- Run all `test_*.py` scripts

## UI Integration
- Connect federation_dashboard.js to backend API for real-time updates
- Use event log for debugging federation events
