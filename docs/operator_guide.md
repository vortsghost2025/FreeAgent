# Operator Guide

## Inspecting Consciousness State
- Use SystemConsciousnessLayer.get_awareness_snapshot() for real-time awareness metrics.
- Adaptive actions are suggested based on system tension and stability.

## Querying Federation Codex
- Use FederationMemoryCodex.get_timeline() and query_history() to review federation history.
- summarize_era() provides narrative summaries for any era.

## Reviewing Myths
- EmergentMythCompiler.compile_from_events() and compile_epoch_myths() generate mythic lore.
- export_myth_codex() outputs myths in markdown or JSON.

## Inspecting Causality Chains
- CrossEpochCausalityRouter.validate_causality_chain() checks for causality integrity.
- trace_causal_lineage() traces event lineage across epochs.

## Monitoring Temporal Negotiations
- TemporalNegotiationEngine.propose_temporal_agreement() and simulate_outcome_over_time() manage time-aware policy changes.
- replay_negotiation_context() replays negotiation history for any federation.

## Example Flow
- An anomaly triggers myth compilation, which updates the codex and influences consciousness state.
- Federation policy changes are simulated and routed across epochs, with causality validated and logged.
