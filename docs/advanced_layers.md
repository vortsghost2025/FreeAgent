# Advanced Subsystems Overview

## Consciousness Layer
Tracks emergent self-awareness by aggregating signals from narrative, anomaly, federation, and persistence.
- API: SystemConsciousnessLayer
- Methods: update_from_signals, get_awareness_snapshot, suggest_adaptive_actions
- Integration: narrative engine, anomaly engine, federation politics, persistent logs

## Temporal Negotiation Engine
Enables federations to negotiate across time using versioned policies and historical context.
- API: TemporalNegotiationEngine
- Methods: propose_temporal_agreement, simulate_outcome_over_time, replay_negotiation_context
- Integration: persistent universe logs, federation politics module

## Federation Memory Codex
Versioned, queryable federation history.
- API: FederationMemoryCodex
- Methods: record_event, get_timeline, query_history, summarize_era
- Integration: persistent logs, documentation layer, narrative templates

## Emergent Myth Compiler
Turns events/anomalies/federation actions into mythic lore.
- API: EmergentMythCompiler
- Methods: compile_from_events, compile_epoch_myths, export_myth_codex
- Integration: narrative engine, anomaly engine, codex, templates

## Cross-Epoch Causality Router
Routes events and negotiations across epochs with causality guarantees.
- API: CrossEpochCausalityRouter
- Methods: route_event, validate_causality_chain, trace_causal_lineage
- Integration: persistent universe states, temporal negotiation, mesh/federation topology
